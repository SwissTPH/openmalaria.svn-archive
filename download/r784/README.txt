Open malaria downloads
----------------------

What's here:

 * linux32.tar.bz2 - Build and required files to run on linux 32-bit systems, assuming a few required libraries are installed
 * macosx.tar.bz2  - Build and required files to run on mac OS X Intel 32-bit systems (requires xerces-c library?)
 * win32.zip       - Build and required files to run on windows 32-bit systems
 * files           - Copy of the above as loose files organized slightly differently
